Constraints
Must remain free.
Must support self-hosting.
Must not require proprietary APIs.
Must work on laptops with limited resources.
Must support local LLMs via Ollama.
Must be modular and extensible.
Must support repositories with thousands of files.
Must provide safe sandbox execution.
Must run entirely offline except for optional GitHub access.
Must maintain privacy by keeping code and data on the user’s machine.
